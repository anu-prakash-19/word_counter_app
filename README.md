# 📝 Word Counter

The **Word Counter** is a simple and interactive web application that allows users to enter text and instantly get the count of words and characters.  
It is useful for students, writers, and professionals who want to keep track of their content length.

---

## 🚀 Features
- Counts **words** in real-time  
- Counts **characters (with and without spaces)**  
- Simple and clean **user interface**  
- Lightweight and **easy to use**

---

## 🛠️ Technologies Used
- **Python (Flask)** – Backend framework  
- **HTML, CSS** – Frontend UI  
- **JavaScript** – For interactive updates (if used)
